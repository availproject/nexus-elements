## API Reference

### Swaps

The `Swaps` component provides cross-chain swaps in either exact-in or exact-out mode with lifecycle callbacks. Both modes are handled in `SwapWidget`.

<CodeCollapsibleWrapper className="mt-4">

```ts title="SwapWidgetProps"
interface SwapWidgetProps {
  onComplete?: (amount?: string) => void; // called when the swap completes
  onStart?: () => void; // called when the swap starts
  onError?: () => void; // called when an error occurs
}
```
