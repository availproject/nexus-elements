## Usage (Exact In)

```tsx showLineNumbers
<Swaps
  exactIn
  onComplete={completeCurrent}
  onError={handleError}
  onStart={handleStart}
  exactInprefill={{
    fromToken: "USDC",
    fromChainID: SUPPORTED_CHAINS.BASE,
    fromAmount: "100",
    toToken: "USDT",
    toChainID: SUPPORTED_CHAINS.ARBITRUM,
  }}
/>
```

## Usage (Exact Out)

<Swaps

## API Reference

### Swaps

The `Swaps` component provides cross-chain swaps in either exact-in or exact-out mode with optional prefill and lifecycle callbacks.

<CodeCollapsibleWrapper className="mt-4">

```ts title="SwapsProps"
interface SwapsProps {
  exactIn?: boolean; // swap mode: exact-in (default: true) or exact-out
  onComplete?: (amount?: string) => void; // called when the swap completes
  onStart?: () => void; // called when the swap starts
  onError?: (message: string) => void; // called on error
  exactInprefill?: {
    // optional prefill for exact-in mode
    fromChainID?: number;
    fromToken?: string;
    fromAmount?: string;
    toChainID?: number;
    toToken?: string;
  };
  exactOutprefill?: {
    // optional prefill for exact-out mode
    toAmount?: string;
    toChainID?: number;
    toToken?: string;
  };
}
```

</CodeCollapsibleWrapper>
