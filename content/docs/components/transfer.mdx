## API Reference

### FastTransfer

The `FastTransfer` component uses the `transfer` function from `@avail-project/nexus-core`. It can be fully customized and styled.

<CodeCollapsibleWrapper className="mt-4">

```ts title="FastTransferProps"
interface FastTransferProps {
  prefill?: {
    token: SUPPORTED_TOKENS; // token to bridge
    chainId: SUPPORTED_CHAINS_IDS; // chain to bridge to
    amount?: string; // amount to bridge
    recipient?: Address; // recipient address to receive the transferred assets
  };
  onComplete?: () => void; // callback when the bridge is complete
  onStart?: () => void; // callback when the bridge starts
  onError?: (message: string) => void; // callback when an error occurs
}
```

</CodeCollapsibleWrapper>
